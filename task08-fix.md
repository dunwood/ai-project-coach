**3. "什么是 Git？"**

```
Git 是一个版本控制工具——你可以把它理解为项目文件的"时光机"。

它能做什么：
- 每次你的项目有进展，Git 帮你保存一个"快照"（叫做 commit）
- 如果改坏了，可以回退到之前任意一个快照
- 记录每次改了什么、什么时候改的

为什么需要它：
AI 编程工具（如 Claude Code）会自动使用 Git 来管理你的代码。每完成一个任务，它会帮你保存一次。如果某个任务做坏了，可以回到上一个好的版本重来。

安装方法：
- Windows：下载安装 https://git-scm.com，一路点"下一步"即可
- Mac：打开终端输入 git --version，如果没有会自动提示安装

安装后你不需要学 Git 命令，AI 编程工具会帮你操作。但知道它是什么、为什么有用就够了。
```

**4. "什么是 GitHub？"**

```
GitHub 是一个代码托管平台——你可以把它理解为"代码的网盘"。

它能做什么：
- 把你电脑上的项目文件备份到云端
- 即使换电脑、重装系统，代码都不会丢
- 可以把项目分享给别人看

为什么需要它：
- 很多部署平台（如 Cloudflare Pages）直接从 GitHub 读取代码并自动上线
- AI 编程工具经常会帮你把代码推送到 GitHub
- 它是程序员世界的"标配"，注册一个不亏

注册方法：
1. 打开 https://github.com
2. 点 Sign up，用邮箱注册（免费）
3. 注册完成后记住你的用户名

你不需要学会 GitHub 的所有功能，只需要有一个账号。后续 AI 编程工具会帮你操作。
```

**5. "什么是 Node.js？需要装吗？"**

```
Node.js 是一个 JavaScript 运行环境——你可以把它理解为很多开发工具的"底座"。

它本身你不会直接用，但很多工具依赖它才能运行：
- Claude Code 需要 Node.js
- 很多网页项目的开发工具需要 Node.js
- npm（Node.js 自带的包管理器）用来安装各种工具

需要装吗：
- 如果你选了 Claude Code → 必须装
- 如果你选了 Cursor 或 TRAE → 暂时不用，但以后可能需要
- 总之建议装一个，迟早用得上

安装方法：
1. 打开 https://nodejs.org
2. 下载 LTS 版本（长期支持版，更稳定）
3. 双击安装，一路点"下一步"
4. 安装完成后打开 PowerShell，输入 node --version，看到版本号就说明成功了
```

**6. "什么是 Cloudflare Pages？"**

```
Cloudflare Pages 是一个网站托管平台——你可以把它理解为"帮你把网站放到互联网上的服务"。

你在电脑上做好的网页，双击 index.html 只有你自己能看到。要让全世界的人都能通过网址访问，就需要把文件"部署"到一个服务器上。Cloudflare Pages 就是做这件事的。

为什么选它：
- 免费（个人项目完全够用）
- 速度快（全球都有服务器节点）
- 在中国大陆可以正常访问（这很重要，很多国外平台不行）
- 可以绑定你自己的域名（比如 xxx.zhexueyuan.com）
- 连接 GitHub 后，每次你推送代码就自动更新网站

怎么用：
1. 注册 Cloudflare 账号：https://dash.cloudflare.com/sign-up
2. 进入 Workers & Pages
3. 创建一个 Pages 项目，连接你的 GitHub 仓库
4. Cloudflare 会自动把你的代码变成一个在线网站

后续 AI 编程工具帮你把代码推到 GitHub 后，Cloudflare 会自动部署，你不需要额外操作。
```
