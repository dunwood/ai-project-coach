# Task 06-fix: 重构任务执行流程

## 问题

当前 tasks.html 的第三步叫"生成首轮开发 Prompt"，只生成 task01 的 prompt，之后直接跳到第四步"开始实战"。用户不知道 task02～task19 怎么执行。

## 目标

把第三步改造成一个**可循环使用的任务执行中心**，用户每完成一个 task 就回来选下一个，直到全部完成。

---

## 修改 tasks.html 的第三步

### 删除原来的"第三步"全部内容

把原来的「第三步：生成首轮开发 Prompt」整个区域删掉，替换为以下新内容：

### 新的第三步：逐个执行任务

```
=== 区域三：执行任务（循环使用） ===

标题："第三步：逐个执行任务"

说明文字（用醒目的提示框样式，青色左边框）：
"这是你的任务执行中心。流程是这样的：
 ① 选择要执行的任务编号 → ② 生成该任务的执行 Prompt → ③ 复制 Prompt 到 AI 编程工具执行 → ④ 完成后回到这里，选下一个任务 → 重复直到全部完成。"

标题："当前要执行哪个任务？"

  按钮行可横向滚动（移动端不换行，用 overflow-x: auto）
  
  下方小字提示："点击按钮选择任务，长按或双击可标记为已完成"
  
  已完成计数："已完成 0/19 个任务"（数字根据标记完成的按钮实时更新）
  
  注意：按钮总数不是固定的，而是根据用户在第二步粘贴的任务清单自动识别。
  识别逻辑：在粘贴的文本中用正则匹配 task01、task02... 或 Task 1、Task 2... 等常见格式，
```

## Prompt 模板

根据选中的任务编号 N 动态生成：

### 当 N = 1 时（首个任务，特殊模板）

```
我要开始做一个新项目，请帮我执行第一个任务。

项目名称：{project_name}
项目简介：{core_goal}
技术栈：{tech_stack}

以下是完整的任务清单：
---
{task_list_content}
---

请执行 task01 的内容。

要求：
1. 只执行 task01，不要做后续任务
2. 完成后告诉我做了什么、创建/修改了哪些文件
3. 如果需要我操作（如浏览器登录等），请先说明

请开始执行。
```

### 当 N >= 2 时（后续任务，通用模板）

```
我正在开发一个项目，现在继续执行下一个任务。

项目名称：{project_name}
项目简介：{core_goal}
技术栈：{tech_stack}

以下是完整的任务清单：
---
{task_list_content}
---

请执行 task{N_padded} 的内容。

要求：
1. 只执行 task{N_padded}，不要做其他任务
2. 注意与之前已完成任务的代码保持一致和兼容
3. 完成后告诉我做了什么、创建/修改了哪些文件
4. 如果发现之前任务的代码有问题需要调整，请先说明再修改

请开始执行。
```

### 变量说明

- `{N_padded}`：两位数补零，如 02、03、19
- `{project_name}`：从 clarify_answers.project_name 读取
- `{core_goal}`：从 clarify_answers.core_goal 读取
- `{tech_stack}`：根据 clarify_answers.platform_type 推断：
  - "网页" → "纯 HTML + CSS + JavaScript，无框架"
  - "微信小程序" → "微信小程序原生开发"
  - "APP" → "待确认技术栈"
  - "还没想好" → "待确认技术栈"
- `{task_list_content}`：从 task_plan.task_list 读取

如果 task_list 为空，显示提示"请先在第二步粘贴任务清单"，生成按钮禁用。

## 任务完成状态存储

在 task_plan 中新增 completed_tasks 数组：

```json
{
  "task_plan": {
    "task_list": "完整任务清单...",
    "first_prompt": "（可以保留或删除，不再单独使用）",
    "completed_tasks": [1, 2, 3]
  }
}
```

用户点击任务按钮标记完成时，更新这个数组并保存到 localStorage。

## 底部操作栏更新

原来的「保存并标记完成」逻辑更新：

- 如果 completed_tasks 包含所有任务 → 按钮文字改为「🎉 全部任务已完成！」，点击弹出庆祝提示
- 如果还有未完成任务 → 按钮文字保持「保存并标记项目完成」，点击时先确认："还有 X 个任务未完成，确定要标记项目完成吗？"
