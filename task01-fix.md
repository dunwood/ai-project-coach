# Task 01-fix: 首页修正 + 产品方向调整

## 方向调整说明

本站定位是**教练/指导**，不是在线 AI 工具。用户在这里学会方法、拿到模板和提示词，然后去站外的大模型和 AI 编程工具里实际操作。

因此：
- 不需要 API Key 输入功能
- 不需要站内 AI 对话功能
- 需求澄清环节改为：引导式表单 + 生成提示词 → 用户复制到站外大模型
- 各页面提供大模型超链接入口（方便用户跳转）
- 删除 js/api.js 文件（不再需要）

## 修正 4：FAQ 全部更新

5 个问题替换为：

**1. "我完全不懂编程，能用吗？"**
```
完全可以。本站会一步步引导你把想法整理成方案，并教你怎么用 AI 编程工具（如 Claude Code、Cursor、TRAE 等）把方案变成产品。你不需要自己写任何代码。
```

**2. "这个网站本身会帮我写代码吗？"**
```
不会。本站是教练，不是代码生成器。它帮你做两件事：第一，把你的想法整理成专业的设计书和任务清单；第二，手把手教你怎么用外部 AI 工具完成开发。所有实际操作都在你自己的电脑和 AI 工具中进行。
```

**3. "需要付费吗？"**
```
本站完全免费。你需要的 AI 工具（如 Claude、ChatGPT、豆包、Kimi 等）大多有免费额度可以起步。本站会在相应环节提供各工具的链接和使用说明。
```

**4. "我的数据安全吗？"**
```
本站所有数据都存储在你自己的浏览器中，不会上传到任何服务器。清除浏览器数据会导致记录丢失，建议定期导出备份。
```

**5. "生成设计书之后怎么办？"**
```
本站会继续教你下一步：如何选择 AI 编程工具、如何安装、如何进入命令行、如何把任务交给 AI 执行、如何验证结果。从想法到上线，全程有指导。
```
