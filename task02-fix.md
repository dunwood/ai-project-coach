## 第一部分：法律声明弹窗内容更新

打开 js/legal.js，找到法律声明弹窗的正文内容，做以下三处修改：

### 修改 1：开头描述

原文中类似：
```
本站致力于帮助零基础用户通过 AI 工具将软件想法变成可执行的项目方案，旨在为用户提供项目规划参考与学习支持。
```

改为：
```
本站致力于帮助零基础用户从软件想法出发，一步步完成项目定义、设计书生成、任务拆解，直到产品上线的全过程指导，旨在为用户提供项目规划参考与学习支持。
```

### 修改 2：外部服务风险

原文中第3条类似：
```
3. 外部服务风险
本站使用第三方 AI 服务（如 DeepSeek）。相关服务的可用性、稳定性...

```
3. 外部服务风险
本站自身不直接调用任何第三方 AI 服务。用户在使用过程中，需自行前往站外 AI 平台（如 Claude、ChatGPT、DeepSeek、豆包、Kimi、通义千问等）进行操作。相关第三方平台的可用性、稳定性、访问速度、政策调整与功能变化，不受本站控制。因第三方平台限制、服务中断、版本变更或其他外部原因导致的使用异常，本站不承担责任。
```

### 修改 3：新增收费相关声明

在「一、声明性质」段落末尾，增加一段：

```
本站提供的访问凭证（注册码），仅作为本在线文档与相关内容的阅读/访问凭证，不构成软件产品销售、预付式消费服务或其他形式的商业功能交付承诺。注册码一经激活，视为已接受本声明全部条款。

## 第二部分：全站页面导航完善

所有二级、三级页面（除 index.html 外），在面包屑导航区域确保有以下两个返回按钮：

### 需要检查和修改的页面

- idea.html
- clarify.html
- design.html
- tasks.html
- workspace.html

### 导航要求

每个页面的顶部（nav 下方、页面标题上方）都要有一个导航条，包含：

```html
<div class="page-nav">
  <a href="index.html" class="nav-link">← 返回首页</a>
  <a href="javascript:history.back()" class="nav-link">← 返回上一页</a>
</div>
```

样式要求：
- 两个链接横排显示，中间用分隔符或间距隔开
- 字号小一号（14px），颜色用二级文字色（#9898a8）
- hover 变为强调色（#00e5ff）
- 移动端两个链接也保持横排，不换行

如果页面已有面包屑导航，把这两个返回按钮整合进去。如果没有面包屑，新增这个导航条。
